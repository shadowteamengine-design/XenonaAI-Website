About This Project

The story behind XenonaAI and why I built it

Who Am I?

I'm Darian Jones (ShadowTeam), a developer passionate about AI and machine learning. I wanted to understand how language models actually work, so I decided to build one from scratch - no shortcuts, no pretrained models.

Why This Project?

Too many people use AI without understanding it. I wanted to learn by doing - downloading datasets, training transformers, building interfaces. This is the result of that journey, shared openly so others can learn too.

What's Next?

This is just the beginning! I'm continuously learning and improving the model. Follow along on GitHub to see the progress, contribute ideas, or even help build features. This is a community project!
